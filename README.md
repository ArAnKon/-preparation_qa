# Interview Q&A Mini-Site

This is a self-contained static mini-site generated from the Markdown archive.

## How to open

Open `index.html` in a browser. No build step, server, or external dependency is required.

## Included

- Dark visual theme
- Section filters
- Full-text search
- Answers stored in `qa-data.js` instead of embedded in `index.html`
- Separate expandable Q&A cards
- Responsive layout
- Local images copied into the `assets` directory
